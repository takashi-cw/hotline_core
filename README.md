# hotline_core
ショートストーリーとZenodo公開論文の原文を保管するリポジトリ。

## 目的
- ショートストーリーを作品単位で管理する
- Zenodo公開論文の原文と付随情報を論文単位で管理する

## ディレクトリ構成
text ... 
hotline_core/
├── stories/
│       
└── papers/
...

※2026/03/08：ディレクトリ構成を変更。


## 運用ルール
- 1作品/1論文ごとにフォルダを作る
- Markdown本文と画像・図表を同じフォルダ配下に置く
- 画像参照は相対パスを使う
- 公開済み論文は `papers/<paper-slug>/meta.yaml` に DOI と URL を記録する
